# MyPharma Admin (React)

Initial React + TypeScript admin app scaffold based on the provided Figma project intent.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## API Documentation

See `docs/api-contract.md` for required backend APIs and payloads.
